## üìä Verificaci√≥n de Consistencia

### **F√≥rmula de Verificaci√≥n**

```
Balance Final = Balance Inicial + Suma(Ingresos) - Suma(Egresos)

Para cada transacci√≥n con abono:
  Actualizaci√≥n Total = Actualizaci√≥n Inicial + Actualizaci√≥n Cierre
  Actualizaci√≥n Total = (Monto * %Abono) + (Monto * %Restante)
  Actualizaci√≥n Total = Monto * (%Abono + %Restante)
  Actualizaci√≥n Total = Monto * 1.0
  Actualizaci√≥n Total = Monto ‚úÖ
```

### **Script de Verificaci√≥n SQL**

```sql
-- Verificar que todas las transacciones completadas sumen correctamente
SELECT
  cd.id,
  cd.monto_destino,
  cd.abono_inicial_monto,
  cd.saldo_pendiente,
  cd.estado,
  (cd.abono_inicial_monto + cd.saldo_pendiente) as suma_verificacion,
  CASE
    WHEN cd.estado = 'COMPLETADO' AND cd.saldo_pendiente != 0
    THEN '‚ùå ERROR: Saldo pendiente no es 0'
    WHEN cd.estado = 'PENDIENTE' AND cd.saldo_pendiente = 0
    THEN '‚ùå ERROR: Pendiente sin saldo'
    WHEN cd.abono_inicial_monto > cd.monto_destino
    THEN '‚ùå ERROR: Abono mayor al total'
    ELSE '‚úÖ OK'
  END as validacion
FROM CambioDivisa cd
WHERE cd.abono_inicial_monto IS NOT NULL
ORDER BY cd.fecha DESC;
```
